# Simple driver for Martin Lightning DMX512 via USB

## Supported products by USB ID
- 0x11be:0xf808

## Features
- Controls DMX channels via ALSA MIDI

## How to build
<ul>
  <li>make PREFIX=/usr # Linux</li>
  <li>make PREFIX=/usr/local # FreeBSD</li>
</ul>

## Dependencies
<ul>
  <li>LibUSB</li>
  <li>ALSA</li>
</ul>
